Send feedback, report issues, or make suggestions directly to the Terrateam team.

## Usage

```
terrateam feedback <message>
```

## What to Include

For the best support experience, include:

- **Issue reports**: Error messages, affected directories, and reproduction steps
- **Feature requests**: Use case description and expected behavior
- **Questions**: Context about your setup and what you're trying to achieve

## Alternative Support Channels

- **GitHub Issues**: [github.com/terrateamio/terrateam/issues](https://github.com/terrateamio/terrateam/issues)
- **Slack Community**: [terrateam.io/slack](https://terrateam.io/slack)
- **Documentation**: [docs.terrateam.io](https://docs.terrateam.io)
- **Email**: support@terrateam.io
