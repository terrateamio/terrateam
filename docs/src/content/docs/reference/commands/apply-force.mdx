---
title: terrateam apply-force
description: Run an apply even if the change has not passed apply requirements
---

To run an apply even if the change has not passed [apply requirements](/reference/configuration/apply-requirements), use the following command:
```
terrateam apply-force <optional-tag-query>
```
[Learn more about our tag queries](/advanced-workflows/tags).
:::caution
This is a dangerous operation. See [access control](/reference/configuration/access-control) to limit access.
:::
